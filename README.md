# 🤟 AI Hand Sign Language Recognition using CNN

A real-time Hand Sign Language Recognition System built using **Python, TensorFlow, Keras, and OpenCV**. The system detects hand gestures through a webcam and classifies them using a Convolutional Neural Network (CNN), enabling fast and accurate gesture recognition.

---

## 🚀 Features

- 📷 Real-time hand gesture recognition using webcam
- 🧠 CNN-based deep learning model for gesture classification
- 🖼️ Image preprocessing (grayscale, resizing, normalization)
- 🔄 Dataset creation and image augmentation utilities
- 📊 Model evaluation with confusion matrix
- ⚡ Fast and responsive real-time predictions

---

## 🛠️ Tech Stack

- Python
- TensorFlow
- Keras

## ⚙️ Installation

Clone the repository

```bash
git clone https://github.com/Sagaruk06/AI-Hand-Sign-Recognition.git
```

Move to the project directory

```bash
cd AI-Hand-Sign-Recognition
```

Install dependencies

```bash
pip install -r requirements_cpu.txt
```

---

## ▶️ Usage

Run the gesture recognition application

```bash
python recognize_gesture.py
```

Train the CNN model

```bash
python cnn_tf.py
```

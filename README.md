# 🤟 Hand Sign Language Recognition using CNN

A real-time Hand Sign Language Recognition System that uses Convolutional Neural Networks (CNN) to classify hand gestures from webcam input. The project leverages TensorFlow, Keras, and OpenCV to perform image preprocessing, model inference, and real-time gesture recognition.

## 🚀 Features

- Real-time hand gesture recognition using webcam
- CNN-based deep learning model
- Image preprocessing and normalization
- Live prediction of hand signs
- User-friendly and efficient recognition pipeline

## 🛠️ Tech Stack

- Python
- TensorFlow
- Keras
- OpenCV
- NumPy

## 📂 Project Structure

```
dataset/
model/
train.py
predict.py
requirements.txt
README.md
```

## ▶️ Getting Started

1. Clone the repository
2. Install dependencies

```bash
pip install -r requirements.txt
```

3. Run the prediction script

```bash
python predict.py
```
